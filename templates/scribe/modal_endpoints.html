Each new Modal account gets ~$1 of free credits. At ~$0.03 per 5-minute note that's about 33 notes per account. When one is exhausted the system automatically switches to the next.

  1. Go to modal.com and create a new account (use a new email).
  2. Install the Modal CLI: pip install modal
  3. Authenticate: modal token new
  4. Deploy the Omni app from the deploy/lightning_speech_api/ folder:
    modal deploy modal_app_omni.py
  5. Copy the URL printed after deploy — it looks like:
    https://<workspace>--wellnest-speech-api-modal-omni-fastapi-app-omni.modal.run
  6. Find your API key in Modal dashboard → Settings → API tokens, or create one.
  7. Paste the URL and key into the Add endpoint form above and click Test endpoint first.

You can paste the full URL with any path (e.g. /health, /transcribe/omni/file) — we extract the base automatically.
