This model converts the nurse's spoken transcript into structured triage fields (BP, HR, SpO2, chief complaint, etc.). To change the model, update SCRIBE_AZURE_OPENAI_DEPLOYMENT in .env.

How it works: When the nurse clicks the mic button on the triage form, the audio is sent to the server in a single request. The server transcribes it via the cloud speech model, then immediately passes the transcript to the extraction model to parse structured fields - both happen server-side before the response returns. No local GPU or MMS model is involved.

Typical latency: ~1.5-3 seconds for a 10-second dictation clip (network + API dependent).
